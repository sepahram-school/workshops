ğŸ’¡ **What is RisingWave?**
RisingWave is a **streaming SQL engine** designed to handle real-time data ingestion, enrichment, and analytics. It integrates seamlessly with messaging systems like **Kafka/Redpanda** and supports:

- **CDC from relational databases** (PostgreSQL) for reference tables
- **Stream joins** and enrichment of transactional data
- **Materialized Views** for real-time analytics
- **Push-based subscriptions** for downstream services
- **Storage of processed outputs** in **RustFS** and **ClickHouse** in **Parquet format**
